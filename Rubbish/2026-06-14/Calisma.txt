PROJE KAPSAMI (Ne teslim edilecek?)
-----------------------------------
Online saat mağazası demosu:
  · Ziyaretçi / üye: ürün gezer → sepete ekler → sahte ödeme → "sipariş tamamlandı"
  · Üyelik: giriş, kayıt, hesabım, geçmiş siparişler, çoklu adres
  · Admin (Saat0604): ürün/sipariş yönetimi (büyük ölçüde hazır — sadece doğrulama)
  · Ek özellik: basit "Size Öneriyoruz" (marka puanına göre öneri)
  · Kurumsal sayfalar: ana sayfa, blog, hakkımızda, iletişim

Teknik: ASP.NET Web Forms 4.8 + ADO.NET + SQL Server + Mirora teması
Çalıştırma: Visual Studio → Rebuild → F5 / IIS Express
  · HTTP:  http://localhost:61529/
  · HTTPS: https://localhost:44317/

yönetimi yapılır."
