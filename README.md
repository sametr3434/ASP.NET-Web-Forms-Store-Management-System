# ASP.NET Web Forms Store Management System

A store management system developed with ASP.NET Web Forms, C#, SQL Server, and Bootstrap. This project was created as a graduation project and is published for portfolio purposes. All sensitive configuration, credentials, and environment-specific settings have been removed before publication.

## Features

- User authentication and authorization
- Product management
- Category management
- Shopping cart
- Order management
- Administrative dashboard
- SQL Server database integration
- Dynamic content management
- Responsive user interface
- Multi-language infrastructure
- Image upload and management

## Technologies

- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET
- Bootstrap
- HTML5
- CSS3
- JavaScript
- jQuery
